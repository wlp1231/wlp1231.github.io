# 加载图像用 cv::imread

  • imread——Mat 对象
  • IMREAD_UNCHANGED (<0)—— 加载原图
  • IMREAD_GRAYSCALE (0)—— 加载灰度图
  • IMREAD_COLOR (>0)—— 加载 RGB 图
  • opencv 支持 JPG、PNG、TIFF 等常见格式
